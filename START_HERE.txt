  2️⃣  MANUAL START (All Platforms)
═══════════════════════════════════════════════════════════════════════════════════

    Step 1: Open Terminal/Command Prompt
    
    Step 2: Navigate to folder
    $ cd attendence
    
    Step 3: Create virtual environment
    $ python -m venv venv
    
    Step 4: Activate it
    
    Windows:
    $ venv\Scripts\activate
    
    macOS/Linux:
    $ source venv/bin/activate
    
    Step 5: Install dependencies
    $ pip install -r requirements.txt
    
    Step 6: Run application
    $ python app.py
    
    Step 7: Open browser
    🌐 http://127.0.0.1:5000

  8️⃣  TROUBLESHOOTING QUICK FIXES
═══════════════════════════════════════════════════════════════════════════════════

    ❌ "Port 5000 already in use"
    ✅ Edit app.py, change port to 5001
    
    ❌ "Camera access denied"
    ✅ Allow camera in browser settings
    ✅ Try different browser
    
    ❌ "Module not found"
    ✅ Run: pip install -r requirements.txt
    
    ❌ "Session expired"
    ✅ Admin needs to start new session
    
    ❌ "Already submitted for this session"
    ✅ That's correct - one submission per session
    
    ❌ "Database locked"
    ✅ Delete attendance.db and restart
